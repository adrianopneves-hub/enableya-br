Perguntas Frequentes

O template gratuito é realmente grátis?

Sim. Instale, use e comprove o valor. Se você gostar e quiser mais poder de análise, pode fazer o upgrade para o Pacote Premium quando estiver pronto. Sem truques, sem necessidade de cartão de crédito para começar.

Como vocês acessam meus dados do sistema Linx?

Nossa plataforma utiliza as integrações e APIs disponíveis do ecossistema Linx para extrair os dados necessários para a geração dos relatórios. Seguimos as melhores práticas de segurança, acessando apenas as informações essenciais para cada template, sem armazenar dados sensíveis.

Isso funciona para o meu setor (farmácia, moda, etc.)?

Sim. Nossos templates são projetados para serem multi-setoriais. As métricas de Giro de Estoque, Orçamento e Inadimplência, por exemplo, são universais para o varejo. Nosso plano "Sob Medida" pode parametrizar qualquer template para as particularidades do seu negócio.

Quais são os templates pagos?

O pacote premium inclui 8 templates: Negociação de dívida, Relatório de inadimplência, Monitoramento de margens, Pipeline de vendas, Análise de rentabilidade por cliente, Acompanhamento de campanhas, Controle de SLA e Previsão de demanda.
